# Student profile assets

Esta carpeta contiene los artefactos de perfil y comportamiento del tutor.

Contenido:
- `STUDY_TUTOR_PROMPT.md` — disciplina principal del tutor
- `config.template.yaml` — plantilla orientativa de configuración

Uso recomendado:
1. Lee `STUDY_TUTOR_PROMPT.md`.
2. Pega ese bloque como base de custom instructions o prompt fijo del tutor.
3. Ajusta `config.template.yaml` solo si de verdad necesitas tocar Hermes.

No copies esta configuración a ciegas en tu Hermes global.
Usa primero la guía de instalación y adapta lo mínimo necesario.
